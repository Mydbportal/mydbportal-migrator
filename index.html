Interactive CLI

Intelligent Workflow

Portal Database Migrator automates the tedious parts of database management. From fetching the data to securely storing it with metadata.

  • ✓ Metadata generation (JSON) with timestamps, checksums, and user info.
  • ✓ Structured storage layout organized by engine and timestamp.
  • ✓ Push/Restore capabilities to replicate data to target servers.

Command Line Interface

Simple, intuitive, and powerful.

Interactive Mode Shell 
$ dbmigrate interactive

1) Init/Add source server
2) Backup all databases
3) List backups
4) Restore a backup
5) Exit

# Just press the number key to proceed
Direct Commands Shell 
# Backup all databases
$ dbmigrate backup --all

# Restore to a target
$ dbmigrate restore \
  --backup ./backups/mysql/db.sql.gz \
  --target target-id

# List available backups
$ dbmigrate list
